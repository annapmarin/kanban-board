# Kanban board with drag & drop

Kanban-style task management web application, developed as a practice for the Web Application Deployment course of the Web Application Development degree. It allows you to organize tasks into status columns, search for them, filter them, and view real-time statistics. All data is saved in the browser, without the need for a server.
