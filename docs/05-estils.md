# Estils i responsive

## Referències
* [Tailwind UI](tailwindui.com/components) -> cards, formularis i layout
* [Dark mode](tailwindcss.com/docs/dark-mode) -> configuració de l'esquema "dark"
* [Colors](tailwindcss.com/docs/customizing-colors) -> paleta slate, indigo, green i amber
* [Tailwind Cheat Sheet](https://nerdcave.com/tailwind-cheat-sheet) -> resum de les classes de Tailwind


## Paleta de colors:
* Fons base: `bg-slate-900`
* Fonts superfície (columnes, aside, filtres): `bg-slate-800`
* Fons elements (targetes, inputs): `bg-slate-700`
* Text principal: `text-slate-100`
* Accent: `text-indigo-400` i `bg-indigo-600`

## Layout
Realitzat amb el Flexbox de Tailwind.

## CSS personalitzat
Algunes classes que el framework no pot gestionar dinàmicament:
* Ocultar el formulari
* Opacitat i escala durant el dragging
* Contorn discontinu per les columnes en fer drag
